DDL| DROP

DROP is used to delete a whole database or just a table.The DROP statement destroys the objects like an existing database, table, index, or view.
A DROP statement in SQL removes a component from a database management system . 

Syntax:

DROP object object_name
Where object is the object to be removed and may be a DATABASE, TABLE, INDEX or VIEW

EXAMPLE CODE

 DROP TABLE Persons;
The above is the sql command to DROP a TABLE named 'Persons'
