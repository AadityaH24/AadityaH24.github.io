DDL| RENAME

SQL RENAME TABLE syntax is used to change the name of a table. Sometimes, we choose non-meaningful name for the table. So it is required to be changed. 

Syntax:

ALTER TABLE table_name   
RENAME TO new_table_name;
Where object is the object to be changed may be a DATABASE, TABLE, COLUMN

EXAMPLE CODE

ALTER TABLE STUDENTS  
RENAME TO ARTISTS;
The above is the SQL command to RENAME a TABLE
