DDL Commands

In SQL, Data Definition Language is also referred to as DDL. DDL commands are used to create and modify the structure of a database and database objects.

Examples of DDL commands:

CREATE    DROP    ALTER

TRUNCATE  COMMENT   RENAME

CREATE

CREATE – is used to create the database or its objects (like table, index, function, views, store procedure and triggers).
